FUTURE_FS_02 — Client Lead Management System (Frontend)
A React.js CRM dashboard for managing client leads built during my Full Stack Web Development Internship at Future Interns.

📌 Task Description
Build a frontend CRM dashboard to manage client leads with features including lead listing, status updates, notes, and secure admin access.

✨ Features

🔐 Admin Login with JWT authentication
📊 Dashboard with lead statistics
📋 Lead table with search & filter
➕ Add new lead with modal form
✏️ Edit existing leads
🔄 Status update (New/Contacted/Converted/Lost)
🗑️ Delete leads
🔍 Search by name or email
📱 Fully responsive — mobile & desktop


🛠️ Tech Stack

React.js — Frontend framework
Axios — HTTP requests
CSS3 — Styling & responsive design
JWT — Authentication

⚙️ Setup

Clone the repo:

bashgit clone https://github.com/durgamahicherla/FUTURE_FS_02-frontend.git
cd FUTURE_FS_02-frontend
npm install

Make sure backend is running on http://localhost:5000
Run:

bashnpm start
